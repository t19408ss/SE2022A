# 課題2 Hash

実行ごとに乱数を指定した個数だけ発生させ、Hash構造に格納する形とした。
（SEEDを変更していないため毎度同じ数が発生する）

`int hash_insert`がHash構造に数値を格納する関数、`int hash_search`が数値の検索を行う関数。
